2. Click the Open button in the top toolbar.

5. Select the files you want to use.
6. Click Extract to save those files to a folder on your computer.

## 💿 Mounting virtual drives
Mounting allows your computer to treat an ISO file like a real CD or DVD.
1. Right-click the PowerISO icon in your system tray.
2. Select Virtual Drive from the menu.
3. Choose Set Number of Drives.
4. Select 1 Drive.
5. Choose Mount Image from the Virtual Drive menu.
6. Pick your ISO file.
7. Open File Explorer to see your new virtual disc drive.
